思路

动态规划，用 dp[i][j] 表示第 i 到第 j 个字符组成的子串是否是回文串。

  • 对于字符串中的第 i 和第 j 个字符，如果 s[i] == s[j] 且 dp[i+1][j-1] == true，那么可得 dp[i][j] == true，否则为 false；
  • 边界条件为 dp[i][i] == true 和 dp[i][i+1] = (s[i] == s[i+1])。
