论文笔记 Oblivious Neural Network Predictions via MiniONN Transformations

发表于 2020-05-26 | 分类于 Papers |
字数统计: 1.3k 字 | 阅读时长 ≈ 5 分钟

Jian Liu, Mika Juuti, Yao Lu, N. Asokan

CCS 2017

https://dl.acm.org/doi/10.1145/3133956.3134056

摘要

托管在云服务中的机器学习模型越来越受欢迎，但存在隐私风险：向该服务发送预测请求的客户端需要披露潜在的敏感信息。在本文中，我们探讨了隐私保护预测问题：在每次预测之后，服务器对客户端的输入一无所知，客户端对模型一无所知。

我们提出了MiniONN，这是第一种将现有的神经网络转换为不经意的神经网络的方法，该网络以合理的效率支持隐私保护预测。与以前的工作不同，MiniONN不需要改变模型的训练方式。为此，我们为神经网络预测模型中常用的操作设计了健忘协议。我们证明了MiniONN在响应延迟和消息大小方面优于现有的工作。通过对标准数据集训练的几种典型神经网络模型的变换，证明了MiniONN的广泛适用性。

关键词：隐私；机器学习；神经网络预测；安全两方计算

贡献

  • 提出了MiniONN，这是第一个可以将任何普通神经网络模型转换为不经意神经网络的技术，而不需要对训练阶段进行任何修改。
  • 为神经网络预测中的常见操作设计了不经意的协议。特别地，作者使非线性函数(例如，Sigmoid和tanh)服从于作者的ONN变换，而精确度损失可以忽略不计。
  • 构建了MiniONN的完整实现，并通过使用它来转换从几个标准数据集训练的神经网络模型来展示其广泛的适用性。特别是，对于从MNIST数据集[38]训练的相同模型，MiniONN的性能明显好于以前的工作[28, 44]。
  • 分析了模型复杂性对转换后的ONN的预测精度和计算/通信开销的影响。讨论了神经网络设计者如何在预测精度和开销之间选择合适的折衷方案。

神经网络

神经网络由一系列层组成。每一层都接收输入并对其进行处理，以产生作为下一层输入的输出。传统上，层被组织成使得最底层接收输入数据(例如，图像或单词)，而最顶层输出最终预测。典型的神经网络1通过首先应用线性变换，然后应用非线性激活函数，以层组的方式处理输入数据。有时还包括池操作来聚合输入组。

下面将从将神经网络转换为ONN的角度简要描述这些操作。

dropout和dropconnect是线性变换的类型，其中乘法使用0-1随机掩码以元素方式完成[30]。

batch normalization是将输出 $\mathbf{y}$ 移至可服从范围的自适应归一化方法[30]。在预测过程中，batch normalization表现为矩阵乘法和加法。

激活函数

神经网络使用数据的非线性变换-激活函数-来建模输入数据和输出预测之间的非线性关系。作者确定了三个常见的类别：

  • 分段线性激活函数。这类函数可以表示为特定范围内的 $n$ 个线性函数的集合，每个类型 $f_i(y) = a_iy + b_i, y \in [y_i, y_{i+1}]$，其中 $y_i$ 和 $y_{i+1}$ 是该范围的下界和上界。此类别包括激活函数：

    恒等函数 (linear)：$f(\mathbf{y}) = [y_i]$

    线性整流函数 (ReLU)：$f(\mathbf{y}) = [{\rm max}(0, y_i)]$

    带泄露线性整流函数：$f(\mathbf{y}) = [{\rm max}(0, y_i) + a {\rm min}(0, y_i)]$

    Maxout ($n$ pieces)：$f(\mathbf{y}) = [{\rm max}(y_1, …, y_n)]$

  • 连续激活函数。连续函数在某一区域上具有高达所需阶数的连续导数。一些常用的平滑激活函数包括：

    Sigmoid (logistic)：$f(\mathbf{y}) = [\frac{1}{1 + e^{-y_i}}]$

    双曲正切 (tanh)：$f(\mathbf{y}) = [\frac{e^{2y_i}-1}{e^{2y_i}+1}]$

    Softplus：$f(\mathbf{y}) = [\log(e^{y_i}+1)]$
